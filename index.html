Taster IFrame Test Harness

Getting Started

The Taster test harness exists to simulate the interaction points between a pilot and being hosted in the Taster IFrame, pilot authors can use it to check their key integration points before raising it with the Taster team to generate a preview link on the full website.

You should edit the source of this page to change the JavaScript configuration in the header to be the URL of your pilot and also the integration mode (16:9 or "full height").

Info Page

By default, people landing on Taster are presented with an info page about the pilot and then invited to try it. Use the tabs above to switch between Info and Try. Switching between the states will send the 'show' and 'hide' events to your pilot.

This harness will also respond to height change requests sent by your application and will initialise it correctly based on the hashbang parameter passed in the URL.
